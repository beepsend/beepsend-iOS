        message:(NSString *)lMessage usedConnection:(BSConnection *)lConnection 
        dataCoding:(NSNumber *)lDataCoding deliveryReport:(BSDLRReport *)lDLR 
        sender:(BSRecipient *)lSender mccmnc:(BSMCCMNC *)lMccmnc price:(NSNumber *)lPrice 
        timestamp:(BSTimestamps *)lTimestamp recipient:(BSRecipient *)lRecipient 
        valid:(NSDate *)lValidTo;
Parameters
lID

- Lookup ID

lBatch

- id and label or null if message was sent with no batch specified

lMessage

- The message body of sent SMS

lConnection

- Connection details

lDataCoding

- 0 for Gsm7 encoded messages. 8 for UCS-2 encoded messages and 4 for binary messages

lDLR

- One of the following: DELIVRD, EXPIRED, REJECTD, UNKNOWN, UNDELIV, FAILED

lSender

- Source

lMccmnc

- Destination mcc mnc

lPrice

- Message price

lTimestamp

- When the message reached Beepsend in Unix time

lRecipient

- Destination

lValidTo

- Until what date and time the message is considered valid for routing

Returns Lookup object
